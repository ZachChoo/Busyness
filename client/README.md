# Frontend web app
A React web application that allows users to conveniently monitor the busyness of different locations.

### Folder Structure
Overall page view are stored in the Views folder, the components that make up these views are in the Components folder.

### To run locally
1. First install dependencies by running `npm install`.
2. Run `npm start`.

### To deploy to Google Cloud Platform
1. Install the Google Cloud CLI: https://cloud.google.com/sdk/docs/install-sdk.
2. Build the project with `npm run build`.
3. Deploy with `gcloud app deploy`.

### Other notes

Check out the `/admin`! The default password is `password`.
